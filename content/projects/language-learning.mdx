# Japanese Language Learning

A personal web app for studying Japanese, available at [study.truongha.com](https://study.truongha.com/).

Built to support my own Japanese learning journey alongside a full-time engineering job — focused on practical, bite-sized study sessions.

## Features

- Hiragana and katakana practice
- Vocabulary drills
- Clean, distraction-free interface

## Tech

Lightweight web stack — optimized for fast load times and mobile use.
